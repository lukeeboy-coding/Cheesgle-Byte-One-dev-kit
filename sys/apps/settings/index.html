Dev kit disclaimer

Huge parts of the phone have been developed, and you probably can find the actual phone if you look. The purpose of this kit is to provide an extremely stripped down version of the system for developing apps before the phone is finalised.

The actual phone is little like this kit and has many more system features, however it isn't too entertaining now with the lack of apps, so I highly encourage you to use this kit to make whatever you desire to be added to the final phone.


Nothing about the actual phone is finalised. If you find the actual phone, please don't share much info on it as the system will change a lot before release.
